** WIKIS

Los "wikis" de GitHub son una funcionalidad que permite crear y mantener 
documentación colaborativa para un proyecto en particular. Un wiki es un
 repositorio especial dentro de GitHub que se utiliza exclusivamente para almacenar 
y editar páginas de documentación.


- Vamos al boton de wikis

- Create first the page

- Save wikis

-  Podemos crear el footer

- Podemos crear nuevas paginas en new page

Cada paginas que se crea, es una url que podemos compartir.

** Referencias entre paginas

Editamos el cuerpo de una pagina y agregamos un enlace, endonde escribiremnos el texto y la URL
la URL es el nombre de la pagina que se ve en la parte derecha.

- Agregar custom sidebarm para el menu

- En cuerpo agregamos los enlaces en el orden que queremos tenerlo

** Insights

n GitHub, "Insights" (Análisis, en español) es una funcionalidad 
que proporciona información y métricas detalladas sobre un repositorio específico.
 Estas métricas y datos estadísticos ayudan a los colaboradores y administradores del
 proyecto a comprender y evaluar el rendimiento, la actividad y el impacto del repositorio.
